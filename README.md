# Keebzord ⌨️

This is the repo with my `keymap.json` and `firmware.hex`, for my keeb.

Specs:

- dz60 rev 3 usb-c
- halo true switches
- kbdfans bamboo case
- enjoypbt 9009 keycaps
- layout almost pure hhkb
